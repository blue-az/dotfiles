# Main LG

LG TV (SSCR2), 4K, 3840x2160, 60Hz

## Specs
- Panel: Unknown (EDID reports "LG TV SSCR2")
- Resolution: 3840x2160 (4K UHD)
- Refresh: 60Hz
- Inputs: HDMI
- Connection: HDMI-A-1 on desktop
- Year: 2020 (per EDID)

## Sway Output
```
output HDMI-A-1 pos 0 0 res 3840x2160
```

## DDC/CI
Does NOT respond to DDC brightness control (ddcutil). Use TV remote for brightness.

## Notes
- EDID reports generic "LG TV SSCR2" identifier (common across LG C1/C2/C3/QNED series)
- Actual model number not embedded in EDID - check back label
- For best results, enable PC Mode or Game Mode on the TV
