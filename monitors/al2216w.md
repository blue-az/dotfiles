# Acer AL2216W

22" LCD Monitor, 1680x1050, 60Hz

## Specs
- Panel: TN, 22" widescreen
- Resolution: 1680x1050 (WSXGA+)
- Refresh: 60Hz
- Inputs: VGA, DVI
- Connection: DP-3 on desktop (via adapter)

## Sway Output
```
output DP-3 pos 0 390 res 1680x1050
```
Positioned to the left of main monitor, vertically centered.

## DDC/CI
Responds to DDC brightness control via ddcutil.
```bash
# Check brightness
ddcutil getvcp 10

# Set brightness (0-100)
ddcutil setvcp 10 50
```

## Notes
- Older monitor, works well as secondary display
- Lower resolution keeps it snappy
- Good for chat windows, terminals, reference docs
